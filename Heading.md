### Heading Tag :
>There are six headings available in HTML.
```
Tag    Describe

<h1> : Largest heading.
<h2> : Second largest heading.
<h3> : Third largest headline.
<h4> : Fourth largest headline.
<h5> : Second small headline.
<h6> : Smallest among all.
```
```
Ex : 
<!DOCTYPE html>
<html>
   <head>
      <title>Heading Tags</title>
   </head>
   <body>
      <h1>0xaman47</h1>
      <h2>0xaman47</h2>
      <h3>0xaman47</h3>
      <h4>0xaman47</h4>
      <h5>0xaman47</h5>
      <h6>0xaman47</h6>
   </body>
</html>
```
