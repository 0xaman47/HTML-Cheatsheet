### link tag : 
```
<nav> : provide navigation link.
<a href = "link"> : linked url
```
```
<a> attributes : 
href : specific url page that link goes to.

target : default browsing used when the href attribute is present.
possible value:-
* _blank
* _self
* _top
* _parent

download : link is use to be used for downloading a resource.

rel : relationship the current document and the href attribute is present.
possible value :-
alternate : give the alternate representations of the current document.
author : give a link to the current document's author.
bookmark : provide the permalink for the nearest ancestor section.
external :Indicates that the referenced document is not part of the same site as the current document.
help : provide link to context sensitive help.
license :  Indicates that the main content of the current document is covered by the copyright license described by the referenced document.
next : Indicates that the current document is a part of a series, and that the next document in the series is the referenced document.
tag : give a tag that applies too the current document.
search : gives a link to a resource that can be used to search through the current document and its related pages.

hreflang : language code of the destination.
